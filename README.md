# DB-script


**code:** :computer: 

```js
bot.command({
name: "ping", 
code: `Pong!` 
})
```

**variables:** :package:

```js
bot.variables({
  Name: "Name",
  Name2: "Value"
})
```

**status:** :bulb:

```js
bot.status({
  text: "Your Text",
  type: "PLAYING",
  time: 12
})
```

# Reminder

To install the package: `npm install dbd.js`
To start it: `node .`
